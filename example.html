Use color squares to control notice box:
  • drag red square to move notice box around the document
  • to change width of the notice box, drag the green square; the notice box height depends on number of text raws
  • drag blue square to a position where you want a new tie to be created
  • number of ties is multiple, you may drag them on a new position or remove by a single click












NoticeBox.js is a jQuery plugin and uses jCanvas to handle html5 canvas element
